Used in PR4.5 in three places: - Create-link modal: inline as the alias prefix (display="prefix") - Edit-URL modal: inline as the alias prefix (display="prefix") - URL list filter: standalone dropdown (display="picker", mode="filter") Built on the existing `.dropdown` primitive (`static/js/dropdown.js`) so keyboard nav + click-outside-to-close + escape handling come for free. Behavior layer in `static/js/components/domain-picker.js` listens for `dropdown:select` events, syncs the hidden input, and fires a `change` event so callers can react with stock DOM APIs.
